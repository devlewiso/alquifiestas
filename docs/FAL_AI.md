# fal.ai — Nota de desarrollo

Esta herramienta NO es parte del stack de Alquifiestas. Se usa ocasionalmente en desarrollo para generar assets visuales de seed (imágenes de categorías, avatares, mockups de productos) que luego se almacenan estáticamente en `apps/web/public/images/`.

**Las imágenes generadas son seeds estáticos. El producto real usa Supabase Storage para fotos subidas por proveedores.**
